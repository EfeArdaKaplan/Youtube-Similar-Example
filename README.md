### EfeTube: Video Platform User Interface

This front-end project is a **learning-focused** exercise created to replicate the user interface (UI) of a popular video platform (YouTube).

The site features a fully mobile-responsive homepage, category filtering, and a detailed sidebar for navigation.

**Key Features:**

* **Dark Mode Support:** Theme feature that saves the user's preference and allows easy switching between light and dark themes.
* **Mobile-Friendly Design:** A clean, responsive layout with a toggleable mobile menu that works seamlessly across all screen sizes.
