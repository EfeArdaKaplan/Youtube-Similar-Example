# 📺 YouTube Interface Clone

This project is a frontend web application developed to replicate the core user interface and layout of YouTube. It serves as a practical example to demonstrate proficiency in modern web design, complex layouts, and responsive structures.

[Youtube Example](https://youtube-similar-example.vercel.app/)

## 🚀 Project Purpose
This repository was created as an educational practice project to improve my front-end development skills. Recreating a globally recognized and structurally complex interface like YouTube provides excellent hands-on experience with CSS Grid, Flexbox, responsive design principles, and DOM manipulation for interactive UI elements.

## 🌟 Key Features
* **Responsive Video Grid:** A fluid and dynamic grid layout that automatically adjusts the number of video cards displayed based on the screen size (desktop, tablet, and mobile).
* **Interactive Navigation:** A functional sidebar and top navigation bar, mimicking the familiar user experience of navigating through different content categories.
* **Modern UI Components:** Includes detailed UI elements such as search bars, user profile icons, category filter chips, and video thumbnails with hover effects.
* **Clean & Semantic Structure:** Built with maintainable and well-organized code, ensuring cross-browser compatibility and optimal performance.

## 🛠️ Technologies Used
* **HTML5:** Semantic markup for structured and accessible web content.
* **CSS3:** Advanced styling using Flexbox and CSS Grid architecture, alongside Media Queries for full responsiveness.
* **JavaScript (ES6+):** Vanilla JavaScript used for handling interactive UI components, such as toggling the sidebar menu or category filters.
